Assignment 3: Create a Simple RSVP Form

Your task: create a simple RSVP form that collects the following information from an imaginary user registering for an imaginary event:

  • Name
  • Email
  • Telephone
  • Meal Choice (choose one)
    • Beef
    • Chicken
    • Fish
    • Vegan
  • Comments
  • A "submit" button

Check out the W3Schools.com HTML Forms tutorial to learn what HTML elements to use to build your form, and use CSS to style your form in the column to the right.

IMPORTANT: Don't forget to use HTML Label elements to ensure your form is accessible to screen readers. Note that the "for" attribute must match the "id" attribute of the form input element.

NOTES:

  • Make sure you use the right input types for special type of data like email and telephone.
  • Don't worry about making the form functional. Set the form attribute to something like: 
    action="/action_page.php"
  • Check this page for example telephone form input validation.
